# 人相兵法 — 觀察總表 v1.2
> 本表列出所有部位的觀察項目，不分維度、不分評分。
> 用途：(1) 知識存檔防止記憶模糊 (2) app 問卷設計的觀察清單基礎。
> 狀態：涵蓋維度 01-13，完整版。

---

## 頭

### 頂骨
| 觀察項目 | 說明 |
|---------|------|
| 左頂骨龜背 | 左頂骨龜背，無凹凸，緩緩向中間隆起 |
| 右頂骨龜背 | 右頂骨龜背，無凹凸，緩緩向中間隆起 |
| 頂骨一致 | 左右頂骨大小凹凸坡度一致 |
| 頂骨不一致 | 左右大小、高低、凸凹不一樣 |
| 頂骨圓 | 頂骨圓而不方（注意：龜背不等於頂骨圓，兩者為獨立觀察條件） |
| 頂骨中線直 | 百會穴為起點，矢狀縫中線向前為直線到髮際中央，沒有向左向右 |
| 頂骨中線斜歪 | 百會穴為起點，矢狀縫中線向前偏左或偏右 |
| 頂骨中央突起 | 正看頭頂明顯一條突起 |

### 枕骨
| 觀察項目 | 說明 |
|---------|------|
| 枕骨圓 | 後腦為圓不為平 |
| 枕骨圓且順 | 枕骨圓，飽滿，無凹凸，無自剋骨，無橫條骨 |
| 枕骨一致 | 枕骨兩側一致，無傾斜 |
| 枕骨不一致 | 左右大小、高低、凸凹不一樣 |
| 枕骨中線直 | 百會穴為起點到枕骨下緣中點為直線，沒有向左向右 |
| 無自剋骨 | 枕骨無自剋骨突起 |
| 無橫條骨 | 枕骨無明顯橫向骨突起 |
| 有自剋骨 | 枕骨後端有一圓或尖突起骨 |
| 有橫條骨 | 枕骨有明顯橫起骨 |

### 華陽骨
| 觀察項目 | 說明 |
|---------|------|
| 左華陽圓 | 左華陽圓隆起，不可尖、平、凹 |
| 右華陽圓 | 右華陽圓隆起，不可尖、平、凹 |
| 左華陽平直 | 左華陽平直，不隆不凹 |
| 右華陽平直 | 右華陽平直，不隆不凹 |
| 左華陽突露 | 左側頭有明顯突起骨 |
| 右華陽突露 | 右側頭有明顯突起骨 |
| 左華陽凹陷 | 左華陽有明顯凹陷 |
| 右華陽凹陷 | 右華陽有明顯凹陷 |
| 華陽順接 | 與頂骨間無太大轉折起伏，順著銜接 |
| 華陽一致 | 左右華陽大小一致 |
| 華陽不一致 | 左右大小、高低、凸凹不一樣 |

**觀察說明：左右華陽各自觀察型相，銜接與一致性為整體觀察。**

### 頭骨整體
| 觀察項目 | 說明 |
|---------|------|
| 頭硬 | 頭按壓感覺較硬，肉偏少 |
| 頭窄 | 百會穴到枕骨下端長度，大於等於，兩側華陽骨上緣間的長度距離 |
| 頭寬 | 百會穴到枕骨下端長度，小於，兩側華陽骨上緣間的長度距離 |

## 上停（額）

| 觀察項目 | 說明 |
|---------|------|
| 額無疵 | 無痕紋痣疤疵陷 |
| 額平 | 正面看額無凹凸、高低不平 |
| 額隆 | 側面看額有隆起弧度 |
| 額立 | 側面看額偏直上而非斜上 |
| 額斜 | 側面看額偏斜上 |
| 額圓 | 正看偏橢圓形、厚度隆起、飽滿感 |
| 額方 | 兩眼角距離約等於眉到髮際距離（形似橫躺長方形） |
| 額長 | 兩眼角距離小於眉到髮際距離 |
| 額寬 | 額頭兩側寬闊 |
| 額大天庭 | 額頭整體隆起（正面和側面皆有隆起感） |
| 額小天庭 | 額頭中間區域（天庭至印堂間）較隆起，兩側不特別寬 |
| 額順 | 眉骨與額頭銜接處無明顯凹槽 |
| 額折 | 眉骨與額頭銜接處有明顯凹槽 |
| 美人尖 | 髮際中央有向下突出 |
| 額緊 | 額骨感明顯，硬度高，且無皺紋 |
| 日月角起 | 上方髮際兩側下方有明顯隆起，左右各一 |
| 日月角開 | 正面看日月角明顯偏兩側，靠近邊城方向 |
| 日月角合 | 正面看日月角較靠近中線 |
| 日月角有肉包 | 日月角肉感為主，骨不凸 |
| 日月角骨感明顯 | 日月角骨較突出 |
| 有輔角骨 | 眉尾上方外側到髮際線有隆起，左右各一 |
| 有佐串骨 | 接近髮際角落轉折處有隆起，左右各一（丘陵、冢墓位） |

## 耳

### 耳型
| 觀察項目 | 說明 |
|---------|------|
| 耳長 | 耳尖端到耳垂長度超過眼到鼻準底的 2/3 |
| 耳短 | 耳尖端到耳垂長度不及眼到鼻準底的 2/3 |
| 耳高 | 耳上緣在眉眼之間、齊眉或以上 |
| 耳立 | 耳上端與耳垂中央一直線，接近垂直，無提耳 |
| 耳提 | 耳上端與耳垂中央一直線朝前，不垂直 |
| 耳圓 | 上半部圓弧完整，中段耳輪較有弧度，下段耳也偏圓，上耳輪不往下翻，耳輪圓無凹陷或波浪 |
| 耳尖 | 上耳輪形尖起 |
| 耳大 | 耳朵大 |
| 耳勢朝上 | 整體耳型朝上長 |
| 耳提強 | 耳上端與耳垂中央一直線朝前約 30 度以上 |
| 招風耳 | 耳和頭顱夾角大於 40 度，整面看明顯超過臉部輪廓線 |
| 耳孔大 | 耳中央盆地範圍等於或超過耳的 1/2 面積（不含耳垂） |

### 輪廓
| 觀察項目 | 說明 |
|---------|------|
| 有輪有廓 | 有明顯的耳輪耳廓，輪廓分明 |
| 輪包廓 | 正面看耳廓不擋耳輪 |
| 輪不包廓 | 耳廓外露，廓不擋輪 |

### 耳質
| 觀察項目 | 說明 |
|---------|------|
| 耳厚 | 耳肉多 |
| 耳堅 | 耳肉不軟，有硬度 |
| 耳硬 | 中段耳質偏硬 |

### 垂珠
| 觀察項目 | 說明 |
|---------|------|
| 有垂珠 | 耳垂大小明顯，且耳垂上有凸起耳珠 |
| 耳垂朝口 | 耳垂朝前翻長朝口 |

### 一致
| 觀察項目 | 說明 |
|---------|------|
| 耳一致 | 左右耳高度、大小、耳型一致 |

**觀察說明：左右耳各自觀察，耳一致為左右合併觀察。**

## 鼻

| 觀察項目 | 說明 |
|---------|------|
| 鼻長 | 山根到鼻準，與鼻準到下巴接近或更長，與印堂到髮際線接近或更長 |
| 鼻高 | 鼻樑（年壽）要高 |
| 鼻直 | 山根到鼻準直線與面部中線重合 |
| 鼻肉 | 鼻肉多於骨 |
| 鼻寬 | 鼻樑寬 |
| 鼻聚 | 準頭要聚不能散，要跟鼻樑寬度差不多 |
| 鼻準圓 | 鼻準不尖，形圓有肉 |
| 鼻接印 | 山根高接印堂，下接年壽 |
| 無起節 | 鼻樑無明顯起節 |
| 鼻順 | 山根、年壽、鼻準側面順接，不折不凹凸 |
| 鼻無肉 | 鼻樑骨感強，明顯看得到鼻骨 |
| 山根細 | 山根寬度明顯短於鼻樑鼻準 |
| 鼻橫張 | 鼻中段年壽寬度接近鼻翼寬度 |
| 骨肉勻稱 | 骨肉平均，肉不多但骨感也不強 |
| 鼻短 | 山根到鼻準長度小於鼻準到下巴，或小於印堂到髮際線 |
| 鼻小 | 鼻翼寬度小於雙眼眼距 |
