## 一、總係數與三大係數的判讀邏輯（Mike 論述）

### 1.1 總係數的意義

Mike 原話：

「先從總係數 跟先天 後天 運氣的係數來比較 當先天係數高於總係數
三項係數各別的係數，可以和總係數作比較，
總係數若高於0.5 表示動靜平衡 基本上是不錯的係數
若低於0.5 表示動靜平衡有待加強 同時也表示 先天後天運氣的係數有偏頗」

### 1.2 偏靜與偏動的行為傾向

Mike 原話：

「那是什麼意思呢
那表示在個性上 會有清楚的傾向 如果偏向靜 會指引到一個人不太容易變動 喜歡坐著等 整體的兵法格局雖然不錯 先天給的素質佳 條件好 不管是原生家庭背景 生長環境 社會地位都有一定基礎 可是面對環境的變數 競爭的變數 就比較不靈活」

「如果偏向動，也是清楚的傾向 指引到一個人在個性上 比較傾向尋找變動的可能 這個變動可能是來自於基本條件與生長環境或者教養過程 相對來說比較不穩定 這個意思不一定是環境變動大 而有可能是生長環境條件並非優於大多數環境 所以需要變動來尋找資源穩固住生長 或者是教養過程中 為了要適切的保護自己 於是持續在變動過程中設法找到自己的生路 也因此導致在競爭環境中 會觀察周遭的變化 也容易因為周遭的變化採取行為的調整 逐漸就會習慣變動 甚至尋找變動的可能性 捨棄自己穩定發展的可能」

### 1.3 三大係數與總係數的比較

Mike 原話：

「總係數跟先天 運氣 後天係數都可以比較出高低
我們通常會先看 總係數跟這三個係數之間的高低關係
會有幾種可能
如果先天高於總係數 表示 運氣 和 後天 一定有一個低於總係數
會指引到說 先天優良 但是 運氣與後天有一個拖垮了總係數
假設是後天 後天係數表示學習方式 應對競爭的戰術 戰略 算略的思維
導致先天的價值無法被充分發揮 這時候就要細看後天指數的維度細節與部位細節
尋找到合適的策略來提升係數 或者找到問題的根源 解釋現在人生進程的卡點或者壞習慣
以此類對」

### 1.4 先天偏動、運氣偏動、後天偏動各自不同

Mike 確認：三層各自的「動」的成因和行為傾向會有不同，不能用同一套描述。（具體差異待後續各層深化討論時補充）

## 二、係數算法選擇（Mike 與 Claude 討論）

### 2.1 兩種方法

**方法 A（維度係數平均）：** 先算每個維度的係數（小/大），再取平均。

**方法 B（小數總和/大數總和）：** 先把所有維度的小數加起來、大數加起來，再做除法。

### 2.2 極端值驗證

Claude 構造驗證案例：先天六維度全部 4:5（拉鋸），運氣三維度全部 9:0（極端）：
- 方法 A 總係數 = 0.533（過 0.50 門檻，判「平衡不錯」）
- 方法 B 總係數 = 0.421（未過 0.50 門檻，判「有待加強」）

### 2.3 Mike 的決定

Mike 原話：

「極端值要納入考量 0.421比較對」

確認採用方法 B。理由：一個有三個維度完全一面倒（9:0）的人，不應該被六個拉鋸維度洗回 0.50 以上。極端維度的影響力應被放大。

## 三、先天與後天的關係（Mike 論述）

### 3.1 資源池與消耗模式

Claude 歸納後 Mike 確認的框架：先天是「資源池」，後天是「怎麼花資源」。

### 3.2 後天四維度動側的共通方向

Mike 原話：

「有接近 從另一個角度來說 用攻 用奇 用虛 用進 其實都是耗費資源的事情 很勞碌 攻城要比守城花數倍得資源才會成功 用奇比用正更要湊齊特殊的資源去執行 行虛比行實更需要面面俱到才有辦法達到效果 行進比行退 更需要氣勢資源跟兵力」

### 3.3 先天不足時攻奇虛進是必要的

Mike 原話：

「當你說沒本錢的拼命者 方向是對的 但對於本錢不夠厚的人來說 其實若要在戰場上生存 則攻奇虛進 就是必要的」

### 3.4 先天充沛時後天更游刃有餘

Mike 原話：

「再回頭看 如果先天是優良的 在攻守奇正虛實進退上 也就更游刃有餘」

### 3.5 動靜不是好壞

Mike 原話：

「正面 和 負面 不是一個好的分法 靜 和 動 並非正面與負面 面相學上雖然有一種依歸是 富貴 面相好 符合相理之美的說法 但我認爲這個當代並不適用
古時候的分法有他的分析原理可以遵循 但不需要硬分成正面負面 面相好 或者面相不好 古時候面相不好基本就人生局限到一個程度 現代社會不是這樣 動有動的希望 靜有靜的好處」

## 五、攻守深度分析（Mike 論述＋老師語錄）

### 5.1 攻守維度間的動靜組合

Mike 原話：

「明星、政治行業，都要攻，要創造話題跟聲量。」

「攻多的人，就會變動、移動、短打、犧牲打，用不斷的變動來調整。」

Mike 原話：

「攻守不一定互補 攻和守是兩種資源條件 跟作戰走向 就如同你說的 有資源有動力去攻 是面相上告訴我們一個人可能選擇的做法 攻守貴在平衡跟彈性 一昧的攻擊 太多攻擊 從頭到上停中庭下停都是攻 那部位本身就不太重要了 也就是守不在他的意圖裡面 需要考慮的是這樣的資源耗費跟疲倦可能帶來的負面效果。
所以攻守的相對關係不是互補 而是要有攻有守 才會有戰術彈性」

### 5.4 攻守與緩急遙相呼應的機制

Mike 原話：

「調整的意思就是針對特定部位造成的個性進行調整 如果我們把它當作部位改善來說 一個維度的特定部位調整一定會影響到其他的維度分數高 所以有可能一個地方調整運氣分數會改善。另外 遙相呼應 跟維度的本質有關 這個之後我們慢慢聊」

## 十、案例五：奇正案例C — 正到底（只有上停奇，鼻正）

## 十六、Claude 的分析歸納（待 Mike 篩選）

### 16.1 虛實四層架構

Claude 歸納的虛實分析框架：
1. 條件層：虛 = 骨露、單薄、凸出、斷裂、塌陷；實 = 肉包骨、平順、穩在該在的位置
2. 面相意義層：「該藏的藏不住」是虛的一致方向
3. 資源意義層：虛是被動流失（無方向性），攻是主動消耗（有方向性）
4. 兵法意義層：虛實是防禦缺口的有無（避實擊虛）

### 16.2 攻守分析框架

Claude 歸納的攻守分析框架：
- 攻守不是互補，是戰術彈性
- 三停是戰略基調，五官是戰術工具
- 攻的門檻普遍很低（任1符合即攻），攻容易被觸發
- 攻的凸露有一部分是武裝工具（vs 虛的凸露是弱點暴露）

### 16.3 奇正分析框架

Claude 歸納的奇正分析框架：
- 奇不是「偏離常規的異常」，是「常規之外多出來的那一支兵力」
- 鼻是奇正的關鍵錨點——鼻正 = 執行回歸正道
- 部位分三組看：自己核心（頭眼鼻）/ 戰場（三停）/ 工具（耳眉口）
- 係數不能脫離部位分布單獨判讀——奇 0.50 可能比奇 0.80 更有效
- 攻配奇順暢，攻配正衝突

### 16.4 跨維度部位比對方法論

Claude 從 Dale 案例歸納：
- 維度告訴你「問題是什麼」，部位告訴你「從哪裡調」
- 找的是「在問題維度裡有潛力變動的部位」
- 調整目標不是翻轉，是微調到平衡

### 16.5 係數判讀的核心原則

Claude 歸納：
- 係數是入口，部位分布才是答案
- 同一個維度同樣的係數，部位分布不同，結論可以完全相反
- 跨維度的配合很重要——奇正不能只看奇正自己，要拉攻守一起看
