# 10 攻守 — 評分標準 v1.4

**維度本質：** 測量面部帶有「銳角、外露、堅硬」的特徵，對應侵略性與好鬥程度。
**攻（動）= 尖露凸硬，有侵略性；守（靜）= 無銳利特徵，溫和被動**

---

## 判斷邏輯

本維度判斷邏輯與其他維度相反——以下列出各部位判「攻（動）」的標準。
**達到門檻 → 該部位判「攻（動）」**
**未達門檻 → 該部位判「守（靜）」**

---

## 評斷順序

頭 → 上停 → 耳 → 眉 → 眼 → 鼻 → 口 → 中停 → 下停

## 最終呈現順序

頭 → 上停 → 中停 → 下停 → 耳 → 眉 → 眼 → 鼻 → 口

---

## 各部位判斷標準

### 1. 頭（5條中任 1 項符合即為「攻」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 頭骨整體 | 頭硬 | 頭按壓感覺較硬，肉偏少 |
| 頂骨 | 頂骨中央突起 | 頂骨中央有明顯一條突起 |
| 枕骨 | 有自剋骨 OR 有橫條骨 | 枕骨有自剋骨，或有橫條骨（任一符合） |
| 左華陽 | 左華陽突露 | 左側頭有明顯突起骨 |
| 右華陽 | 右華陽突露 | 右側頭有明顯突起骨 |

**記錄格式範例：**
- 5條皆無 → **頭守**
- 任1條符合 → **頭攻**

---

### 2. 上停/額（3條中任 1 項符合即為「攻」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 額 | 美人尖 | 髮際中央有向下突出 |
| 額 | 額緊 | 額骨感明顯，硬度高，且無皺紋 |
| 額 | 日月角骨感明顯 AND（大天庭 OR 小天庭） | 日月角明顯突出且骨感強，且額頭有天庭隆起（大天庭或小天庭，任一）（2項全中→攻） |

**記錄格式範例：**
- 3條皆無 → **上停守**
- 任1條符合 → **上停攻**

---

### 3. 耳（左耳 5 條任 1 符合即為攻，右耳同）
**左耳或右耳任一為攻 → 耳攻**
**左右耳皆守 → 耳守**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左耳 | 耳高 | 耳上緣在眉眼之間、齊眉或以上 |
| 左耳 | 輪不包廓 | 耳廓外露，廓不擋輪 |
| 左耳 | 耳尖 | 上耳輪形尖起 |
| 左耳 | 耳硬 | 中段耳質偏硬 |
| 左耳 | 耳勢朝上 | 整體耳型朝上長 |
| 右耳 | 耳高 | 耳上緣在眉眼之間、齊眉或以上 |
| 右耳 | 輪不包廓 | 耳廓外露，廓不擋輪 |
| 右耳 | 耳尖 | 上耳輪形尖起 |
| 右耳 | 耳硬 | 中段耳質偏硬 |
| 右耳 | 耳勢朝上 | 整體耳型朝上長 |

**記錄格式範例：**
- 左右耳皆無 → **耳守**
- 任一耳有1條符合 → **耳攻**

---

### 4. 眉（左眉 4 條中 3 條以上符合即為攻，右眉同）
**左眉或右眉任一為攻 → 眉攻**
**左右眉皆守 → 眉守**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左眉 | 有鷹角 | 眉 2/3 處上緣有明顯轉折，類似三角形 |
| 左眉 | 眉毛稀少 | 無眉毛或眉毛稀疏 |
| 左眉 | 質硬 | 眉毛粗硬不柔順 |
| 左眉 | 有揚 | 眉尾往上 |
| 右眉 | 有鷹角 | 眉 2/3 處上緣有明顯轉折，類似三角形 |
| 右眉 | 眉毛稀少 | 無眉毛或眉毛稀疏 |
| 右眉 | 質硬 | 眉毛粗硬不柔順 |
| 右眉 | 有揚 | 眉尾往上 |

**記錄格式範例：**
- 左右眉皆不足3條 → **眉守**
- 任一眉 3條以上符合 → **眉攻**

---

### 5. 眼（左眼 3 條任 1 符合即為攻，右眼同）
**左眼或右眼任一為攻 → 眼攻**
**左右眼皆守 → 眼守**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左眼 | 眼鉤 | 眼頭有鉤（前眥鉤曲如鳥嘴） |
| 左眼 | 眼大 AND 眼圓 AND 眼尾朝上 | 眼大且偏圓形且眼尾朝上（3項全中→攻） |
| 左眼 | 眼睛凸 | 眼球大較突出眼瞼 |
| 右眼 | 眼鉤 | 眼頭有鉤（前眥鉤曲如鳥嘴） |
| 右眼 | 眼大 AND 眼圓 AND 眼尾朝上 | 眼大且偏圓形且眼尾朝上（3項全中→攻） |
| 右眼 | 眼睛凸 | 眼球大較突出眼瞼 |

**記錄格式範例：**
- 左右眼皆無 → **眼守**
- 任一眼有1條符合 → **眼攻**

---

### 6. 鼻（4條中任 1 項符合即為「攻」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 鼻 | 鼻高 | 鼻樑（年壽）要高 |
| 鼻 | 有起節 | 鼻樑有起節，突露 |
| 鼻 | 鼻無肉 | 鼻樑骨感強，明顯看得到鼻骨 |
| 鼻 | 山根細 | 山根寬度明顯短於鼻樑鼻準 |

**記錄格式範例：**
- 4條皆無 → **鼻守**
- 任1條符合 → **鼻攻**

---

### 7. 口（5條中任 1 項符合即為「攻」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 口 | 嘴角朝上 | 嘴角朝上 |
| 口 | 見齒 | 放鬆時口不緊閉且看得到齒 |
| 口 | 唇垂珠 | 上嘴唇與人中間有小顆突起 |
| 口 | 唇凸 | 嘴唇往外突出或翹起明顯 |
| 口 | 唇薄 | 上下唇皆薄 |

**記錄格式範例：**
- 5條皆無 → **口守**
- 任1條符合 → **口攻**

---

### 8. 中停（滿分 7 分，攻的部位數 ≥ 4 即為「攻」）
**中停計分依賴眉眼鼻顴各自的攻守判斷結果匯總，不重新計算細項**

| 項目 | 條件 |
|------|------|
| 左眉 | 左眉為攻（4條中3條以上符合） |
| 右眉 | 右眉為攻（4條中3條以上符合） |
| 左眼 | 左眼為攻（3條任1符合） |
| 右眼 | 右眼為攻（3條任1符合） |
| 鼻 | 鼻為攻（1項以上符合） |
| 左顴 | 顴高、顴無肉、顴小（任1符合） |
| 右顴 | 顴高、顴無肉、顴小（任1符合） |

**顴的判斷條件（任1符合即為攻）：**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左顴 | 顴高 | 顴骨位置高，靠近眼下緣（Q1 = 顴高且隆） |
| 左顴 | 顴無肉 | 顴骨感明顯覺得堅硬，或顴骨是小塊突起（Q3 = 顴無肉/少肉） |
| 左顴 | 顴小 | 顴骨明顯但是面積小（Q2 = 顴小/尖） |
| 右顴 | 顴高 | 顴骨位置高，靠近眼下緣（Q1 = 顴高且隆） |
| 右顴 | 顴無肉 | 顴骨感明顯覺得堅硬，或顴骨是小塊突起（Q3 = 顴無肉/少肉） |
| 右顴 | 顴小 | 顴骨明顯但是面積小（Q2 = 顴小/尖） |

**記錄格式範例：**
- 7部位均攻 → **中停攻 7分**
- 4部位攻 → **中停攻 4分**（剛好過門檻）
- 3部位攻 → **中停守 3分**

---

### 9. 下停（5個部位，攻的部位數 ≥ 3 即為「攻」）
**口的屬性依賴口的個別攻守判斷結果**

| 項目 | 條件 |
|------|------|
| 口 | 口為攻（1項以上符合） |
| 人中 | 人中窄：人中凹下部分窄，或為一直線 |
| 地閣 | 地閣長、地閣短、地閣窄尖、地閣無肉（任1符合） |
| 左頤 | 頤露尖 OR 頤短（任1符合） |
| 右頤 | 頤露尖 OR 頤短（任1符合） |

**記錄格式範例：**
- 5部位均攻 → **下停攻 5分**
- 3部位攻 → **下停攻 3分**（剛好過門檻）
- 2部位攻 → **下停守 2分**

## 各部位門檻總覽

| 部位 | 判「攻」的門檻 |
|------|------------|
| 頭 | 5條任1符合 |
| 上停 | 3條任1符合 |
| 耳 | 左右耳各5條任1符合，任一耳攻即攻 |
| 眉 | 左右眉各4條中3條以上，任一眉攻即攻 |
| 眼 | 左右眼各3條任1符合，任一眼攻即攻 |
| 鼻 | 4條任1符合 |
| 口 | 5條任1符合 |
| 中停 | 7部位攻的數量 ≥ 4 |
| 下停 | 5部位攻的數量 ≥ 3 |

## 整體計分與係數

### 維度屬性判定
9個部位各自判出「攻」或「守」：
- **攻的部位數 > 守的部位數 → 維度屬性為「動」**
- **守的部位數 > 攻的部位數 → 維度屬性為「靜」**
