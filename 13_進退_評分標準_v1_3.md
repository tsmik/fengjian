# 13 進退 — 評分標準 v1.3

**維度本質：** 測量五官長度或寬度是否「過短、過小或過窄」，對應面對利益時的克制力。
**進（動）= 見機即行事，拼運氣；退（靜）= 深謀遠慮，伺機而動**

---

## 判斷邏輯

本維度判斷邏輯與 10 攻守、11 奇正相同——以下列出各部位判「進（動）」的標準。
**達到門檻 → 該部位判「進（動）」**
**未達門檻 → 該部位判「退（靜）」**

---

## 評斷順序

頭 → 上停 → 耳 → 眉 → 眼 → 鼻 → 口 → 中停 → 下停

## 最終呈現順序

頭 → 上停 → 中停 → 下停 → 耳 → 眉 → 眼 → 鼻 → 口

---

## 各部位判斷標準

### 1. 頭（滿分 5 分，門檻：3 分以上即算「進」）

| 小部位 | 觀察項目 | 說明 | 分數 |
|--------|---------|------|------|
| 頭骨整體 | 頭窄 | 百會穴到枕骨下端長度，大於等於，兩側華陽骨上緣間的長度距離 | 2 |
| 枕骨 | 枕骨不一致 OR 有自剋骨 OR 有橫條骨（任1） | 枕骨左右大小高低凸凹不一樣，或有自剋骨，或有橫條骨（任1） | 1 |
| 左華陽 | 左華陽突露 OR 左華陽凹陷（任一） | 左側頭有明顯突起骨，或左華陽凹陷（任一符合） | 1 |
| 右華陽 | 右華陽突露 OR 右華陽凹陷（任一） | 右側頭有明顯突起骨，或右華陽凹陷（任一符合） | 1 |

**記錄格式範例：**
- 5/5 → **頭進 5分**
- 3/5 → **頭進 3分**（剛好過門檻）
- 2/5 → **頭退 2分**

---

### 2. 上停/額（1 條符合即為「進」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 額 | 額方 | 兩眼角距離約等於眉到髮際線中央距離（額寬和高相近，形似正方） |

**記錄格式範例：**
- 符合 → **上停進**
- 不符合 → **上停退**

---

### 3. 耳（左耳 3 條任 1 符合即為進，右耳同）
**左耳或右耳任一為進 → 耳進**
**左右耳皆退 → 耳退**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左耳 | 耳短 | 耳尖端到耳垂長度不及眼到鼻準底的 2/3 |
| 左耳 | 耳高 | 耳上緣在眉眼之間、齊眉或以上 |
| 左耳 | 招風耳 | 耳和頭顱夾角大於 40 度，整面看明顯超過臉部輪廓線 |
| 右耳 | 耳短 | 耳尖端到耳垂長度不及眼到鼻準底的 2/3 |
| 右耳 | 耳高 | 耳上緣在眉眼之間、齊眉或以上 |
| 右耳 | 招風耳 | 耳和頭顱夾角大於 40 度，整面看明顯超過臉部輪廓線 |

**記錄格式範例：**
- 左右耳皆無 → **耳退**
- 任一耳有1條符合 → **耳進**

---

### 4. 眉（左眉 3 條任 1 符合即為進，右眉同）
**左眉或右眉任一為進 → 眉進**
**左右眉皆退 → 眉退**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左眉 | 眉毛質硬 | 眉毛粗硬不柔順 |
| 左眉 | 眉短 | 眉短不及目 |
| 左眉 | 眉毛短 | 眉毛單根長度較短 |
| 右眉 | 眉毛質硬 | 眉毛粗硬不柔順 |
| 右眉 | 眉短 | 眉短不及目 |
| 右眉 | 眉毛短 | 眉毛單根長度較短 |

**記錄格式範例：**
- 左右眉皆無 → **眉退**
- 任一眉有1條符合 → **眉進**

---

### 5. 眼（左眼 1 條符合即為進，右眼同）
**左眼或右眼任一為進 → 眼進**
**左右眼皆退 → 眼退**

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 左眼 | 眼長 | 眼長度等於或大於眼距 |
| 右眼 | 眼長 | 眼長度等於或大於眼距 |

**記錄格式範例：**
- 左右眼皆無 → **眼退**
- 任一眼符合 → **眼進**

---

### 6. 鼻（3 條任 1 符合即為「進」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 鼻 | 鼻肉 | 鼻肉多於骨 |
| 鼻 | 鼻短 | 山根到鼻準長度小於鼻準到下巴，或小於印堂到髮際線 |
| 鼻 | 鼻小 | 鼻翼寬度小於雙眼眼距 |

**記錄格式範例：**
- 3條皆無 → **鼻退**
- 任1條符合 → **鼻進**

---

### 7. 口（2 條任 1 符合即為「進」）

| 小部位 | 觀察項目 | 說明 |
|--------|---------|------|
| 口 | 口大 | 開大合小 |
| 口 | 口適中 | 口寬過鼻翼，寬度在兩瞳孔距離之內 |

**記錄格式範例：**
- 2條皆無 → **口退**
- 任1條符合 → **口進**

---

### 8. 中停（滿分 7 分，進的部位數 ≥ 4 即為「進」）
**中停計分依賴眉眼鼻顴各自的進退判斷結果匯總，不重新計算細項**

| 項目 | 條件 |
|------|------|
| 左眉 | 左眉為進（3條任1符合） |
| 右眉 | 右眉為進（3條任1符合） |
| 左眼 | 左眼為進（眼長符合） |
| 右眼 | 右眼為進（眼長符合） |
| 鼻 | 鼻為進（3條任1符合） |
| 左顴 | 顴無肉：顴骨感明顯覺得堅硬，或顴骨是小塊突起（Q3 = 顴無肉/少肉） |
| 右顴 | 顴無肉：顴骨感明顯覺得堅硬，或顴骨是小塊突起（Q3 = 顴無肉/少肉） |

**記錄格式範例：**
- 7部位均進 → **中停進 7分**
- 4部位進 → **中停進 4分**（剛好過門檻）
- 3部位進 → **中停退 3分**

---

### 9. 下停（5個部位，進的部位數 ≥ 3 即為「進」）
**口的屬性依賴口的個別進退判斷結果**

| 項目 | 條件 |
|------|------|
| 口 | 口為進（2條任1符合） |
| 人中 | 人中短：人中短於食指寬 |
| 地閣 | 地閣窄尖（Q4 = 地閣窄尖） |
| 左頤 | 頤削 AND 頤露尖（2項全中→進） |
| 右頤 | 頤削 AND 頤露尖（2項全中→進） |

**記錄格式範例：**
- 5部位均進 → **下停進 5分**
- 3部位進 → **下停進 3分**（剛好過門檻）
- 2部位進 → **下停退 2分**

---

## 各部位門檻總覽

| 部位 | 判「進」的門檻 |
|------|------------|
| 頭 | 滿分5分，3分以上 |
| 上停 | 1條符合 |
| 耳 | 左右耳各3條任1符合，任一耳進即進 |
| 眉 | 左右眉各3條任1符合，任一眉進即進 |
| 眼 | 左右眼各1條符合（眼長），任一眼進即進 |
| 鼻 | 3條任1符合 |
| 口 | 2條任1符合 |
| 中停 | 7部位進的數量 ≥ 4 |
| 下停 | 5部位進的數量 ≥ 3 |

---

## 整體計分與係數

### 維度屬性判定
9個部位各自判出「進」或「退」：
- **進的部位數 > 退的部位數 → 維度屬性為「動」**
- **退的部位數 > 進的部位數 → 維度屬性為「靜」**

### 進退係數計算

| 進（部位數） | 退（部位數） | 屬性 | 係數 |
|--------|---------|------|------|
| 0 | 9 | 靜 | 0.00 |
| 1 | 8 | 靜 | 0.13 |
| 2 | 7 | 靜 | 0.29 |
| 3 | 6 | 靜 | 0.50 |
| 4 | 5 | 靜 | 0.80 |
| 5 | 4 | 動 | 0.80 |
| 6 | 3 | 動 | 0.50 |
| 7 | 2 | 動 | 0.29 |
| 8 | 1 | 動 | 0.13 |
| 9 | 0 | 動 | 0.00 |

**係數意義：**
- 越接近 1.0 = 進退越平衡
- 越接近 0 = 越偏向單一極端
